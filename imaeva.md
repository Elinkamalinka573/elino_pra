## 1. Описание проекта
Данный проект представляет собой учебное веб-приложение, разработанное для демонстрации навыков работы с системой контроля версий Git и верстки веб-страниц. Целью работы является освоение полного цикла разработки: от инициализации репозитория до проведения код-ревью и документирования.

## 2. Технологический стек
* Язык программирования: C# (WPF, .NET Framework)
* Система контроля версий: Git / GitHub
* Среда разработки: Visual Studio 2022

## 3. Структура проекта
* imaeva/ — исходный код приложения (логика, разметка XAML).
* imaeva.sln — файл решения Visual Studio.
* imaeva.md — файл описания проекта (документация).

## 4. Результаты практики

### 4.1. Работа с Git
1. Инициализация: Создан локальный репозиторий, выполнена привязка к удаленному репозиторию на GitHub.
2. Ветвление: Организована работа в ветках. Создана ветка develop для внесения изменений, что позволило изолировать процесс разработки от стабильной версии в main.
3. Слияние: После проверки корректности кода выполнено слияние (merge) ветки develop в основную ветку main.

### 4.2. Проведение Code Review
Был проведен анализ фрагмента кода MainWindow.xaml.cs. 
* Объект анализа: Логика управления состоянием кнопки возврата (BtnBack).
* Результат: Произведена замена свойства Visibility.Hidden на Visibility.Collapsed. Это позволило добиться динамического перестроения интерфейса и исключить появление пустых зон в макете при отсутствии истории переходов.

### 4.3. Обратное проектирование алгоритма
Выполнено восстановление логики работы функции навигации.
* Алгоритм: Проверка возможности возврата в MainFrame (CanGoBack) -> Присвоение статуса видимости кнопке (Visible/Collapsed) -> Обновление заголовка окна в зависимости от активного контента.

## 5. Запуск
Для запуска приложения необходимо открыть файл imaeva.sln в среде Visual Studio и выполнить компиляцию проекта (F5).
